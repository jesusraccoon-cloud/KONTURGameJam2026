# Khruchevka — Solo Indie Horror Project

 indie horror project developed in Unity using C#.

The project is an atmospheric VHS/PS1-style horror game inspired by post-Soviet apartment blocks, analog video aesthetics and urban horror.  
The main focus is exploration, tension, interactive environment, anomalies and simple survival-horror mechanics.

## Current Features

- First-person exploration
- Raycast-based interaction system
- Door interaction system
- Interactive objects and collectibles
- VHS cassette collectible mechanic
- Basic UI using TextMeshPro
- AI patrol system based on Unity NavMesh
- Level blockout of apartment and stairwell areas
- Horror atmosphere setup
- VHS/PS1-inspired visual direction
- Git/GitHub version control workflow

## Technologies

- Unity
- C#
- Git / GitHub
- ProBuilder
- NavMesh
- TextMeshPro
- Visual Studio Code

## Project Status

The project is currently in active development as a solo indie game and technical portfolio project.  
Core gameplay systems are being developed and tested step by step.
